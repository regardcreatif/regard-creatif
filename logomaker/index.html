 Guide rapide

1
Choisissez un modèle
Faites défiler les modèles. Tapez un mot-clé dans la recherche pour filtrer par style ou secteur. Cliquez sur une vignette pour l'ouvrir dans l'éditeur.
2
Personnalisez dans l'éditeur
Double-cliquez sur un texte pour le modifier. Utilisez le panneau de droite pour changer couleurs, polices et tailles. Ajoutez formes, lignes ou images.
3
Gérez vos calques
L'onglet Calques (panneau gauche) liste tous les éléments. 🔓 verrouille, 👁 masque, ⋮⋮ réordonne par glisser-déposer.
4
Exportez votre logo
Cliquez sur Exporter en haut de l'éditeur. Choisissez PNG, PNG 2× haute résolution, ou SVG vectoriel (idéal pour impression).
5
Sauvegardez vos créations
Cliquez sur 💾 dans l'éditeur. Votre projet apparaît dans Mes créations sur l'accueil. Tout est sauvegardé localement sur votre appareil.
