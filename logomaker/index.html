 Guide rapide

1
Choisissez un modèle
Faites défiler les modèles. Tapez un mot-clé pour filtrer. Cliquez sur une vignette pour l'ouvrir.
2
Personnalisez dans l'éditeur
Double-tapez sur un texte pour le modifier. Utilisez le panneau droite pour couleurs, polices et tailles.
3
Gérez vos calques
L'onglet Calques liste tous les éléments. Maintenez ⋮⋮ pour réordonner.
4
Exportez votre logo
Cliquez sur Exporter. Choisissez PNG, PNG @2x, ou SVG vectoriel.
5
Sauvegardez
Cliquez sur 💾. Votre projet apparaît dans Mes créations sur l'accueil.
💡
Gestes dans l'éditeur
2 doigts : pincer pour zoomer / pan pour se déplacer. 1 doigt : déplacer un élément. Double-tap : éditer le texte.
