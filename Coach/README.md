# Le Coach - IA Marketing Digital 🧠

Coach IA 100% hors-ligne avec 100+ concepts marketing, correcteur de fautes, et contenu adapté à Haïti.

## Fichiers inclus

| Fichier | Description |
|---------|-------------|
| `index.html` | Application complète (tout-en-un, aucune dépendance) |
| `style-coach.css` | Styles du coach (pour intégration) |
| `bouton-coach.js` | Script du bouton Coach (pour intégration) |
| `manifest.json` | PWA manifest (installable sur mobile) |

## Usage rapide

**Option A — Fichier tout-en-un :**
Ouvrez simplement `index.html` dans votre navigateur. Aucune installation.

**Option B — Intégrer le Coach sur votre site existant :**
```html
<!-- Dans <head> -->
<link rel="stylesheet" href="style-coach.css">

<!-- Avant </body> -->
<script src="bouton-coach.js"></script>
```
Le bouton 🧠 Coach sera automatiquement ajouté à votre barre de navigation.
